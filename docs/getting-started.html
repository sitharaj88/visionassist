Getting Started

Set up VisionAssist in your local environment

Try Live Demo First

Prerequisites

Before you begin, make sure you have the following installed:

  • Node.js 18.0 or higher (Download)
  • npm 9.0 or higher (comes with Node.js)
  • Git for cloning the repository
  • Google Gemini API Key (Get one free)

Installation

Step 1: Clone the Repository

git clone https://github.com/sitharaj88/visionassist.git cd visionassist

Step 2: Install Dependencies

npm install

This will install all required packages including Next.js, React, and the Google AI SDK.

Step 3: Configure Environment Variables

Create a .env.local file in the project root:

# .env.local GEMINI_API_KEY=your_api_key_here

Important: Never commit your API key to version control. The .env.local file is already in .gitignore.

Step 4: Start Development Server

npm run dev

Open http://localhost:3000 in your browser.

Getting Your Gemini API Key

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the generated key
  5. Paste it into your .env.local file

Note: The free tier includes generous usage limits sufficient for development and personal use.

Building for Production

Build the Application

npm run build

Creates an optimized production build in the .next folder.

Start Production Server

npm start

Deployment

Vercel (Recommended)

The easiest way to deploy Next.js apps.

  1. Push your code to GitHub
  2. Import project at vercel.com/new
  3. Add GEMINI_API_KEY in Environment Variables
  4. Deploy!

Netlify

Alternative hosting with great performance.

  1. Connect your GitHub repository
  2. Set build command: npm run build
  3. Set publish directory: .next
  4. Add environment variables

Docker

Containerized deployment for any platform.

docker build -t visionassist . docker run -p 3000:3000 visionassist

Troubleshooting

Camera Not Working

Make sure you're accessing the app via HTTPS or localhost. Cameras require a secure context.

API Key Error

Verify your .env.local file exists and contains a valid GEMINI_API_KEY. Restart the dev server after changes.

Voice Commands Not Working

Web Speech API requires microphone permission. Check browser settings and ensure you're using a supported browser (Chrome, Edge, Safari).

Build Errors

Try deleting node_modules and .next folders, then run npm install again.

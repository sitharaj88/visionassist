See the World Through AI

VisionAssist empowers visually impaired individuals with AI-powered visual assistance. Describe scenes, read text, identify objects, and navigate spaces - all through voice commands.

5 Powerful Analysis Modes

Each mode is optimized for specific visual assistance needs

Scene Description

Get detailed descriptions of your surroundings with spatial awareness - left, right, near, far positions.

Read Text (OCR)

Extract and read aloud text from documents, signs, labels, menus, and any printed material.

Object Identification

Identify products, items, and objects with detailed information including brands and usage.

Navigation Assistance

Get guidance for navigating spaces with clock positions (12 o'clock ahead), obstacles, and pathways.

Color Detection

Identify colors with specific names - perfect for clothing, food items, and everyday objects.

Voice Control

Fully hands-free operation with 12+ voice commands. Just speak to capture and analyze images.

Quick Start

Get up and running in under 5 minutes

1. Clone the repository

git clone https://github.com/sitharaj88/visionassist.git cd visionassist

2. Install dependencies

npm install

3. Set up environment

cp .env.example .env.local # Add your GEMINI_API_KEY

4. Start development server

npm run dev

Open http://localhost:3000 in your browser.

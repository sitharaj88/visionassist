Features

Everything you need for AI-powered visual assistance

5 Analysis Modes

Each mode is optimized with specialized AI prompts for specific visual assistance needs.

Scene Description

Provides comprehensive descriptions of environments with spatial awareness. Uses directional language (left, right, center) and distance indicators (near, far) for better orientation.

Best for: Understanding surroundings, rooms, outdoor spaces

Keyboard: Press 1

Read Text (OCR)

Extracts and reads text from images with high accuracy. Handles documents, signs, labels, menus, packaging, and handwritten notes.

Best for: Documents, signs, menus, labels, mail

Keyboard: Press 2

Object Identification

Identifies specific items with detailed information including brand names, product types, and usage suggestions.

Best for: Products, groceries, medications, household items

Keyboard: Press 3

Navigation Assistance

Provides guidance for moving through spaces safely. Uses clock positions (12 o'clock = straight ahead) for directions and identifies obstacles, doorways, and pathways.

Best for: Indoor navigation, finding exits, avoiding obstacles

Keyboard: Press 4

Color Detection

Identifies colors with specific, descriptive names (not just "blue" but "navy blue" or "royal blue"). Perfect for matching clothes or identifying items by color.

Best for: Clothing, food freshness, organizing by color

Keyboard: Press 5

Voice Control

Fully hands-free operation with natural voice commands.

Available Voice Commands

Command Action
"capture" or "take picture"Capture current camera view
"scene"Switch to Scene Description mode
"read"Switch to Read Text mode
"identify"Switch to Object Identification mode
"navigate"Switch to Navigation mode
"color"Switch to Color Detection mode
"stop"Stop current speech
"repeat"Repeat last description
"history"Read history count
"last result"Repeat most recent analysis
"continuous on"Enable auto-capture every 5 seconds
"continuous off"Disable continuous mode
"help"List available commands

Additional Features

Analysis History

Stores up to 10 recent analyses with thumbnails and timestamps. Quickly review or re-read previous results.

Keyboard: Press H

Quick Actions

One-tap shortcuts for common tasks: "What's Around Me?", "Read This Now", and "Help Me Navigate".

Keyboard: Press Q

Continuous Mode

Automatically captures and analyzes every 5 seconds for real-time assistance while moving.

Demo Mode

Try the app offline with 5 pre-loaded scenarios covering all analysis modes. Perfect for learning or demonstrations.

Copy & Share

Easily copy analysis results to clipboard or share via native sharing on mobile devices.

Settings Persistence

Your preferences (theme, font size, high contrast, auto-speak) are saved and restored automatically.

Offline Demo
